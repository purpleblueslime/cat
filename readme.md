# About

a tiny tamagotchi-like frog pet built using **`next.js`**, **`scss`**, **`tauri`**. 🍥

# Build yourself

You'll need **`rust`** and **`bun`**/**`node.js`** installed

## Install deps

```css
> bun install
```

## Run app `dev`

```css
> bun run tauri dev
```

## Build release

```css
> bun run tauri build
```
